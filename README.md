# Cerradura-RFID
Cerradura Funcional con RFID y Arduino, con futura autenticación de dos pasos
---

### Requisitos
· Placa de desarrollo (Arduino UNO o NANO) <br>
· Arduino IDE <br>
· Modulo RC522 <br>
· Tarjetas RFID <br>
· Cerradura electrica de 12v <br>
